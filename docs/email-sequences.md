## Configuração Técnica

### Headers recomendados
```
From: Figurinha Fácil <oi@figurinhafacil.com.br>
Reply-To: suporte@figurinhafacil.com.br
List-Unsubscribe: <mailto:unsubscribe@figurinhafacil.com.br>
```

### Segmentação
- `new_user`: Signup < 7 dias
- `active`: Login últimos 7 dias
- `has_matches`: Matches > 0
- `no_stickers`: Figurinhas cadastradas = 0
- `inactive_7d`: Sem login 7-30 dias
- `inactive_30d`: Sem login 30+ dias
- `high_value`: Muitas trocas confirmadas

### A/B Tests sugeridos
1. Subject lines: Emoji vs. sem emoji
2. CTA: "Ver matches" vs. "Encontrar trocas"
3. Timing: Manhã vs. noite
4. Personalização: Nome vs. sem nome
