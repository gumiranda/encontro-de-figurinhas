### Full Description
```
## O problema

O álbum da Copa do Mundo 2026 tem 980 figurinhas. 
Completar comprando só pacotinhos? Prepare-se para gastar R$ 5.000+.

A solução óbvia é trocar. Mas:
- Grupos de WhatsApp são bagunçados
- É difícil encontrar quem tem exatamente o que você precisa
- Onde trocar com segurança?

## A solução

Figurinha Fácil resolve isso em 3 passos:

1. **Cadastre suas figurinhas**
   Digite os números das repetidas e faltantes. 2 minutos.

2. **Encontre matches automáticos**
   O sistema cruza sua lista com milhares de colecionadores.
   Veja quem tem o que você precisa — e está perto de você.

3. **Combine a troca**
   Escolha um ponto de troca verificado, combine via WhatsApp, troque.

## Features

✓ Match automático por proximidade
✓ 800+ pontos de troca verificados em 150+ cidades
✓ Sistema de check-in e reputação
✓ Notificações de novos matches
✓ 100% gratuito para colecionadores

## Por que construímos isso?

Somos colecionadores desde criança. A cada Copa, a mesma frustração: 
pilha de repetidas, lista de faltantes, ninguém pra trocar.

Decidimos resolver de vez.

## Stack

Next.js 15, Convex (real-time), Clerk, Vercel

## Roadmap

- [x] Match automático
- [x] Mapa de pontos
- [x] Sistema de check-in
- [ ] App nativo (em breve)
- [ ] Integração com álbum virtual Panini

---

Feito com ❤️ por colecionadores, para colecionadores.
Bora completar esse álbum!
```

## Possíveis Perguntas e Respostas

**P: Funciona fora do Brasil?**
R: Por enquanto só no Brasil, onde está a maior base de colecionadores. 
   Mas o sistema pode expandir para outros países no futuro!

**P: É grátis mesmo?**
R: Sim, 100% grátis para colecionadores. Monetizamos com features 
   premium opcionais (em breve) e parcerias.

**P: E se não tiver ninguém na minha cidade?**
R: Cadastre-se assim mesmo! Você ajuda a crescer a rede e será 
   notificado quando houver colecionadores perto.

**P: Como vocês verificam os pontos de troca?**
R: Combinação de dados públicos, validação da comunidade, e 
   sistema de check-in que confirma atividade real.

**P: Qual a diferença para grupos de WhatsApp?**
R: Match automático! Você não precisa ficar postando lista. 
   O sistema cruza automaticamente e te avisa.

**P: Vocês têm app mobile?**
R: O site é 100% responsivo e funciona como app. 
   Apps nativos estão no roadmap para após a Copa.
